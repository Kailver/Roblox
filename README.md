# PrisonCore

Custom-coded Paper prison core aimed at running a modern classic prison server with room to expand.

## Included systems
- A-Z rank ladder and Prestige 1-10 progression.
- Cash economy, tokens, player-to-player payments, and sell-all support.
- Virtual backpack storage and autosell toggling.
- FAWE/WorldEdit-backed prison mines with automatic reset support.
- Per-item prison pickaxes with persistent stats, unique enchant builds, and dynamic lore.
- 26 custom enchants including both mining and visual proc enchants.
- GUI enchant menu for upgrading enchants while holding a prison pickaxe.
